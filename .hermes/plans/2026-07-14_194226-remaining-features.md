# PromptMate 剩余功能清单

> **For Hermes:** 后续按优先级逐项以 TDD 实现；每个独立功能验证后单独 Conventional Commit 并推送。

**目标：** 基于当前可运行的 React + Tauri 词库工作台，补齐 MVP 和完整规划中的真实业务能力。

**当前基线：** 已有 Windows EXE/MSI、词条卡片选择、双语静态拼装、来源标签、Vitest/ESLint/Prettier/Cargo Test；当前词库仍是 12 条 TypeScript 演示数据，大部分按钮和筛选仅有界面。

**统计口径：** 原任务清单 T20–T56 共 37 个工程任务。目前 2 项完整、9 项部分完成、26 项尚未实现；预计仍有约 85%–90% 的实际功能工作量。

## 推荐下一开发顺序

1. 数据 Schema + JSONL 正式词库加载与校验。
2. 图片/视频切换、真实分类筛选和本地搜索。
3. 灵感篮删除、清空、排序、复制和结果编辑。
4. 收藏、最近使用、模板和推荐组合。
5. JSONL/CSV 导入导出（用户词条管理已完成）。
6. API 凭据与可选 AI 辅助。
7. 图片转提示词与语义匹配。
8. 悬浮窗与自动粘贴。
9. 视频结构化编辑。
10. E2E、CI、干净系统安装测试与正式发布。
